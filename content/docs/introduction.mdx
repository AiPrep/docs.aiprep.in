# Introduction

DocsTech is a modern documentation platform built for developer teams. Ship beautiful docs that stay in sync with your codebase.

## Why DocsTech?

- **MDX-powered** — Write docs in Markdown with embedded React components.
- **Git-native** — Content lives alongside your code. Every change is versioned.
- **Fast by default** — Static generation with incremental builds. Pages load instantly.
- **Fully themeable** — Ship on-brand docs with custom themes and layouts.

## Quick Example

Create an MDX file in your content directory:

```mdx
---
title: Getting Started
description: Set up your first project in under 5 minutes.
---

# Getting Started

Welcome to the docs!
```

That's it. DocsTech picks it up, adds it to the sidebar, and renders it as a fully styled page.
