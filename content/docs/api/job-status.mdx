# Job Status

Poll the status of a video generation job.

```
GET /api/v1/jobs/{job_id}
```

<ApiPlayground
  method="GET"
  url="https://api.aiprep.in/api/v1/jobs/your-job-id"
  headers='[{"key":"X-API-Key","value":""}]'
/>

## Path Parameters

- `job_id` (string, required) — The unique job identifier returned from [/generate](/api/generate)

## Headers

- `X-API-Key` (string) — Your API key (use this **or** Authorization)
- `Authorization` (string) — JWT token as `Bearer <token>`

## Response Fields

- `job_id` (string, required) — Unique job identifier.
- `status` (string, required) — `"pending"`, `"processing"`, `"completed"`, `"failed"`, or `"cancelled"`.
- `progress` (integer, default: `0`) — Progress percentage (0–100).
- `current_step` (string | null) — Description of the current processing step.
- `video_url` (string | null) — Final video URL (available when `status` is `"completed"`).
- `video_path` (string | null) — Internal video storage path.
- `thumbnail_url` (string | null) — Video thumbnail URL.
- `duration` (number | null) — Video duration in seconds.
- `file_size_mb` (number | null) — Video file size in megabytes.
- `has_narration` (boolean | null) — Whether the video includes narration.
- `pipeline_variant` (string | null) — Processing pipeline variant used.
- `segments` (array | null) — Segment summaries with `index`, `label`, and `duration`.
- `error` (string | null) — Error message if `status` is `"failed"`.
- `retry_count` (integer | null) — Number of retries attempted.
- `created_at` (string, required) — ISO 8601 timestamp of job creation.
- `updated_at` (string, required) — ISO 8601 timestamp of last update.
- `steps` (array | null) — Detailed breakdown of processing steps, each with `name`, `status`, `progress` (0–100), and `estimated_seconds`.
- `estimated_time_remaining` (integer | null) — Estimated seconds remaining.
- `elapsed_time` (integer | null) — Seconds elapsed since job started.
- `feedback_prompt` (object | null) — Feedback prompt data for the UI.
- `preview_video_url` (string | null) — URL for a preview video (partial render).
- `preview_segments_complete` (integer | null) — Number of preview segments completed.
- `preview_segments_total` (integer | null) — Total number of preview segments.

> Poll every 2-5 seconds until `status` is `"completed"` or `"failed"`. For real-time updates without polling, use the [SSE Stream](/api/stream) endpoint instead.
